Car Mileage Data Visualization

This project tracks my real-world driving data in Wisconsin, including miles per fill, MPG, gas prices, and total fill cost. I wrote a Java script to convert my raw log file into a JSON dataset, then used Chart.js to visualize trends directly in the browser.

The charts below are rendered from data.json, which is generated from my handwritten mileage log. Each fill-up becomes a data point that can be explored over time to spot patterns in fuel efficiency and gas prices.
